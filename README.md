# CBP Rupiah — GitHub Pages Ready

Website statis edukasi CBP Rupiah untuk GitHub Pages.

## Isi halaman

- `index.html` — Beranda
- `cbp.html` — Cinta, Bangga, Paham Rupiah
- `sejarah-rupiah.html` — Sejarah Rupiah
- `qris.html` — QRIS
- `kuis.html` — Kuis interaktif CBP
- `tentang-website.html` — Tentang Website

## Cara upload ke GitHub Pages

1. Buat repository baru di GitHub, misalnya `cbp-rupiah`.
2. Upload semua file dan folder dari ZIP ini ke repository tersebut.
3. Pastikan `index.html` berada langsung di root repository, bukan di dalam folder tambahan.
4. Buka **Settings > Pages**.
5. Pilih **Deploy from a branch**.
6. Pilih branch `main` dan folder `/root`.
7. Klik **Save** dan tunggu beberapa menit.
8. Website akan aktif di alamat seperti:
   `https://username.github.io/cbp-rupiah/`

## Cara mengganti logo Bank Indonesia

1. Siapkan file logo berformat PNG.
2. Ubah nama file menjadi `logo-bi.png`.
3. Upload ke folder:
   `assets/img/logo-bi.png`
4. Refresh website. Logo akan tampil otomatis. Jika file logo belum ada, website menampilkan fallback teks “BI”.

## QR Code

Setelah link GitHub Pages aktif, gunakan link tersebut untuk membuat QR Code.
