# Rupiah 3D Specimen Viewer

Website statis siap upload ke GitHub Pages.

## Isi
- `index.html`
- `assets/css/style.css`
- `assets/js/app.js`

## Cara upload ke GitHub Pages
1. Buat repository baru atau buka repository lama.
2. Upload seluruh isi folder ini ke root repository.
3. Pastikan `index.html` berada langsung di root, bukan di dalam folder tambahan.
4. Buka `Settings` → `Pages`.
5. Pilih `Deploy from a branch`.
6. Pilih branch `main` dan folder `/root`.
7. Klik `Save`.

## Catatan
Visual uang di website ini adalah **specimen edukasi**, bukan uang asli, bukan alat pembayaran, dan bukan replika detail uang asli.
